**radius**: base de datos utilizada por Radius, se encarga de la autorización del usuario, es aquí donde se almacena la contraseña, contiene el username, contraseña, además de otros campos importantes para el radius.



**sdn\_policy\_v2:** base de datos utilizada para almacenar la información correspondiente al usuario, como su información personal, roles, carrera, cursos y también para almacenar información acerca de los recursos presentes en nuestra red como justamente los recursos asociados a los cursos (ej: web), portal general, biblioteca, entre otros.
